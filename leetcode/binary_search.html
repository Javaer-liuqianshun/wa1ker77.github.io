# 二分查找

# 33. Search in Rotated Sorted Array

假设一个按升序排序的数组以一个不确定的轴旋转。如 [0,1,2,4,5,6,7] 旋转成 [4,5,6,7,0,1,2]称为旋转数组。

在旋转数组中找到一个给定值元素，返回下标索引，没有找到返回 -1。

数组中没有重复元素，并用时间复杂度 O(n) 的算法实现。

Example1：

Input: nums = [4,5,6,7,0,1,2], target = 0
Output: 4

Example2：

Input: nums = [4,5,6,7,0,1,2], target = 3
Output: -1

实现：

// a表示数组，n表示数组大小，target表示给定值
public int search(int[] a, int n, int target) {

    int low = 0;
    int high = n - 1;

    while (low <= high) {
        int mid = low + ((high - low) >> 1);
        if (a[mid] == target) {
            return mid;
        }
        if (a[mid] < a[high]) { // a[mid] < a[high] 当前 mid 在旋转数组小的一块（即后一块）
            if (target > a[mid] && target <= a[high]) { // 如果是给定值元素在旋转数组小的一块
                low = mid + 1;
            } else {
                high = mid - 1;
            }
        } else { // a[mid] >= a[high] 当前 mid 在旋转数组大的一块（即前一块）
            if (target >= a[low] && target < a[mid]) {// 如果是给定值元素在旋转数组大的一块
                high = mid - 1;
            } else {
                low = mid + 1;
            }
        }
    }
    return -1;
}
