# 声明

2018年初在知乎看见一篇专栏，标题为程序员你为什么这么累？，并配上图作为背景图，瞬间被吸引。点进去后仅读完导读就发现晓风轻老师点出来的问题就是我的日常开发操作，一口气读完后备受打击，感觉自己连编程门都没有入，仅仅是代码的搬运工。

2018年6月在项目不紧张时，刷知乎又看见程序员你为什么这么累？专栏，点击进入后发现专栏已经连载完了，顺其自然重新开始读了一片，再一次阅读收获也是巨大的。并有想法把文章转载在自己笔记网站上。

2018年6月底程序员你为什么这么累？专栏文章已全部转载在wa1ker77笔记 - 编程规范上。

很感谢晓风轻老师。
